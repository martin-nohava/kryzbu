Client
=============

Klient je konsolová aplikace sloužící pro přístup ke vzdálenému úložišti. Klient komunikuje se `serverem <server.html>`_ a to vždy stylem dotaz odpověď. 
Komunikaci vždy zahajuje klient. Jak klientskou aplikaci používat vysvětluje kapitola `Začínáme <gettingStarted.html>`_.
